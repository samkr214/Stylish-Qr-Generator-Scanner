About

This Stylish QR Code Generator is a modern web application built using HTML, CSS, and JavaScript. It allows users to generate high-quality QR codes with customizable colors, sizes, and even logos embedded in the center.

The project also includes advanced features like QR scanning (Image & PDF), download options in PNG and PDF formats, and a clean, responsive user interface with dark mode support for a better user experience.

This project is designed to demonstrate practical implementation of front-end development, canvas rendering, file handling, and API integration in a real-world use case.
